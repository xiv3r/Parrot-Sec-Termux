# Auto install
```
apt update && pkg install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Parrot-Sec-Termux/refs/heads/main/install.sh | sh && clear && parrot
```
# Login
```
parrot
```
# Logout
```
exit
```
# Uninstall 
```
uninstall-parrot
```
