# Parrot-Sec-Termux
Parrot Sec with GUI in Termux

# Auto install
```
apt update && apt install wget -y && wget -qO- https://raw.githubusercontent.com/xiv3r/Parrot-Sec-Termux/refs/heads/main/install.sh | sh
```
- VNC XFCE Desktop Environment
```
wget -qO- https://raw.githubusercontent.com/xiv3r/Parrot-Sec-Termux/refs/heads/main/gui.sh | sh
```

# Run
```
parrot
```
