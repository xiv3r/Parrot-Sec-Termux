# Auto install
```
apt update && apt install wget -y && wget -qO- https://raw.githubusercontent.com/xiv3r/Parrot-Sec-Termux/refs/heads/main/install.sh | sh && parrot
```
# Login
```
parrot
```
# Logout
```
exit
```
# Uninstall 
```
uninstall-parrot
```
